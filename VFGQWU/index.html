What Is Binary?

Binary is a number system that uses only two digits: 0 and 1. It’s the fundamental language of computers. Every piece of data — text, images, sounds, and instructions — is ultimately represented in binary. Computers use electrical signals to process information: 1 means "on", and 0 means "off".

For example, the binary number 1010 represents the decimal number 10. Each digit (called a bit) corresponds to a power of 2, starting from the right.
